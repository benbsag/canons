      data & sync

      Sync keeps your cellar matched on every device, automatically. Set it up once and anything you add or edit shows up on your other devices on its own — no files to move. On a new device, open this menu, paste the sync code from a device that's already linked, then tap link this device.

      Export / import is the manual version. Export wines saves your whole cellar to a single file you can keep as a backup or send to someone. Import wines loads that file back in. Nothing updates by itself — you do it whenever you want a snapshot.

      The difference: both move your wines around — sync does it for you continuously, export/import is a one-off copy you trigger by hand. Use sync for everyday across-device use; run an export now and then as a safety backup.
